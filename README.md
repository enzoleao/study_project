

## Sobre
API com foco para estudos utilizando o framework
[Nest](https://github.com/nestjs/nest) 


## Features
- Paginação com filtros
- Autenticação JWT Token

## Futures Features
- Permissões para acesso as rotas
  
## Instalação

```bash
$ yarn install
```

## Running the app

```bash
$ sudo docker-compose up -d --build

# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```
